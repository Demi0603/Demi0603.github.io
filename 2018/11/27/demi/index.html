静态网站

静态网站开发的定位

*一开始：网站几乎是静态
    *CERN，Tim Bernede Lee：超链接、world wide web
*动态网站
    *CGI：
        *实践建议：下载Putty、Fterm等终端工具，连接访问bbs.pku.edu.cn北大未名网站，
    *脚本：ASP、PHP
    *半编译型：Java、ASP.Net
    *相对而言执行速度慢，配置稍复杂
*个别情境重新关注静态网站
    *在线手册

静态网站的生成工具

*主要特点
    *一般是指用第三方程序语言开发，基于用户输入的内容快速生成静态网站
    *一般也会支持主题
    *大多数静态网站生成工具也会支持同步到服务器的选项
*常用系统
    *Hexo：Node.js开发，支持用Markdown写内容
    *Sphinx：Python开发，支持用Restul Text写内容
    *MkDoc：Python开发，支持Markdown写内容

Node.js

*Javascript
    *最全能的语言
    *用Javascript实现后端开发的一个代表：Node.js
    *用Javascript实现客户端：
        *示例：Visual Studio Code
        *技术：Ecletron

Node.js

*下载安装Node.js
    *方法一：下载Windows Installer
    *方法二：下载Windows binary解压，并将路径放入PATH变量
        *echo %PATH%查询
    *检测node.js安装状态
        *node -v
        *npm -v

安装hexo

*hexo是一个node.js包
*包安装方法
    *node.js中的包管理器：npm（node package manager）
    *npm install 包名（参数）
    * npm install hexo-cli -g

cmd命令中
D：加回车，跳到D盘
dir 可以列出当前的文件夹下的文件
cd +文件夹名，改变目录，进入到此文件夹
也可以 cd+文件夹名\文件夹 路径
